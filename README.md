# Smart Lock System with Fingerprint Recognition

This project is a smart lock system that works with fingerprint authentication. The system recognizes authorized users, records failed login attempts, and sends alarms and notifications in suspicious situations. User management and entry/exit logs can also be viewed via the web panel.

## Tools
- Microcontroller (ESP32 / Arduino)
- Fingerprint sensor
- Solenoid lock
- Buzzer and LED
- Wi-Fi
- SD card / Cloud data recording

## Features
- Fingerprint registration/verification
- Trial limit and “lock-in”
- Entry and exit log and viewing via mobile and web interfaces
- Self-alerting, mobile and email alerts/notifications
